在laravel中支持百度小程序等相关功能

## 安装

Via Composer

``` bash
$ composer require yangze/laravelsmartprogram
```

``` bash
$ php artisan vendor:publish --provider="yangze\LaravelSmartprogram\LaravelSmartprogramServiceProvider"
```
